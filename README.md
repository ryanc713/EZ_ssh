### Installation

1. Clone the repo
   git clone https://github.com/ryanc713/EZ_ssh.git
2. cd EZ_ssh
3. chmod +x EZ_ssh.sh
4. Open in your favorite editor and replace all the necessary information with your own.
5. Rename when saving and closing if you wish.
6. Run script



<!-- USAGE EXAMPLES -->
## Usage

Rename this script to the domain name of the Server you will use this to login to. For example, iamryancook.com. Then fill in all the details with your own, and invoke the script.

./iamryancook.com

_For more examples, please refer to the [Documentation](https://example.com)_
